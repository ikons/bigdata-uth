# Contributing to the bilingual repository

This repository keeps the original Greek material and the English translations side by side.

## Naming convention

- Greek Markdown guides keep their original name, for example `README.md`
- English Markdown guides use `README.en.md`
- English Word guides use `.en.docx`
- Code files remain under the same path; inline teaching comments are written in English

## Recommended maintenance workflow

1. Update the Greek source material first
2. Mirror the corresponding changes in the English `.en.md` file
3. Regenerate the `.en.docx` handouts from the English Markdown when needed
4. Keep command examples, paths, and screenshots aligned across both versions

## Practical rule

Treat the Markdown files as the canonical source and the `.docx` files as exported teaching handouts.
